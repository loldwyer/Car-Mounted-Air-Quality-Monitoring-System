Hardware Overview

This system uses an ESP32 with the following sensors:

  • SCD30: CO₂, Temperature, Humidity
  • SPS30: PM1.0, PM2.5, PM10
  • MiCS-4514: NO₂
  • DS3231 RTC for timestamps
  • SD card for local logging
  • Blynk + ThingSpeak for remote access
